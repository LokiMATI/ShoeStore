# Как запустить проект
Для начала, необходимо в СУБД MS SQL загрузить последний Backup базы данных.

После этого можно запускать различные проекты и проверять их работоспособность.

## Что не реализовано
- Автоинкрементация новых значений в БД и импорт данных в xls и csv (SQL Server Management Studio 22 не предоставляет на дайнный момент часть функционала, позволяющая решить данные вопросы. Старые версии SMSSQL не удалось установить)
- Рефакторинг кода
- Хэширование пароля с помощью BCrtypt
